nurikabe
========
Nurikabe solver.
Bryan Clair 2015

Requires networkx python module.

Built to work with arbitary graphs, though ASCII I/O is only available for square grids.

Example
-------

`python nurikabe.py examples/example.txt`
or
`python nurikabe.py < examples/example.txt`

where `examples/example.txt` might contain

    ...2.
    .....
    ....3
    .....
    .4...
